# Thomas Lepère — Portfolio website

Static website ready for GitHub Pages.

## Publish
1. Create a new GitHub repository, for example `portfolio`.
2. Upload the contents of this folder to the repository root.
3. In **Settings → Pages**, select **Deploy from a branch**.
4. Select the `main` branch and `/ (root)`.
5. GitHub will provide the public URL.

For a personal root site, name the repository `<github-username>.github.io`.

## Edit
- Text and sections: `index.html`
- Colours and layout: `styles.css`
- Project filtering and pop-ups: `script.js`
- Figures: `assets/`
